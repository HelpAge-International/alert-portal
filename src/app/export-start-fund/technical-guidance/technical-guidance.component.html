Start Fund Project Application Form


Technical Guidance

The following guidance has been designed for use with the Project Reporting Form contained within the next three worksheets.

General Guidelines
  1. Before you begin completing the form, please save a new copy of the file with the following format: "Start Fund Project Application - (your agency name) (crisis name) Response ((project lead name)). For example: Start Fund Project Application - ACF Somalia Response (Jane Doe)
  2. All Project Application Forms should be submitted by or on behalf of the the Project Lead.
  3. Please enter all text in Arial, size 10 font. The comment and track changes function is now enabled.
  4. In cells whose response requires input of numbers, only whole numbers are accepted.
  5. In the Project Activities worksheet, ten lines have been provided for listing different types of activities.
    1. In the first column, please contain your activities to these ten lines and be succinct when listing each individual activity (2-5 words each).
    2. In the second column, state the specific output expected from each activity. If a given activity has multiple outputs, separate them using a semi-colon (;).
    3. In the third column, give a specific Key Performance Indicator (KPI) for each activity in terms of the short or medium term effects the activities are expected to have. If multiple activities have the same KPI, repeat the KPI for each applicable activity.
  6. In the Project Budget worksheet, please list your project inputs by sector. The total will automatically calculate.
