Start Fund Project Application Form


Technical Guidance

The following guidance has been designed for use with the Project Reporting Form contained within the next three worksheets.

General Guidelines
  1. Before you begin completing the form, please save a new copy of the file with the following format: "Start Fund Project Application - (your agency name) (crisis name) Response ((project lead name)). For example: Start Fund Project Application - ACF Somalia Response (Jane Doe)
  2. All Project Application Forms should be submitted by or on behalf of the the Project Lead.
  3. Please enter all text in Arial, size 10 font. The comment and track changes function is now enabled.
  4. In cells whose response requires input of numbers, only whole numbers are accepted.
  5. In the Project Activities worksheet, ten lines have been provided for listing different types of activities.
    1. In the first column, please contain your activities to these ten lines and be succinct when listing each individual activity (2-5 words each).
    2. In the second column, state the specific output expected from each activity. If a given activity has multiple outputs, separate them using a semi-colon (;).
    3. In the third column, give a specific Key Performance Indicator (KPI) for each activity in terms of the short or medium term effects the activities are expected to have. If multiple activities have the same KPI, repeat the KPI for each applicable activity.
  6. In the Project Budget worksheet, please list your project inputs by sector. The total will automatically calculate. In addition, five lines have been provided for listing individual items costing more than Â£1,000 each. If you need to report on more than five items, please use the Additional Comments section to do so.

Specific Guidelines
  1. Needs assessment: You were not required to complete a separate needs assessment for this project, but please provide details of the most relevant assessment to which this project responded.
  2. Lessons Learned: The M&E Team will use your answer to identify learning that can be shared and further developed and may follow-up with you directly for more details in some cases.
  3. Visual Documentation: You are encouraged to submit visual documentation (e.g. photos, videos) with your reporting form as this is highly valued for case studies and communications.
  4. Beneficiary Influence: If beneficiaries have not been substantially involved in the project, please explain why the context prevented this.
  5. The one-percent: See technical guidance on the application form (worksheet 1) for more information about the 1% learning budget and the simple process for accessing and reporting on it.
  6. Peer-review: Reports will be submitted to the Learning and Evaluation Committee for peer-review by Start Fund agencies, and the M&E Team may contact you directly with any follow-up questions.
